ContentFetcherFactory

ContentFetcherFactory

A factory class for creating content fetcher instances based on different API types. This class abstracts the creation logic for various content fetchers, allowing for easy extension and maintenance. It currently supports the Peacock API, but other APIs can be added as needed.

Constructor

new ContentFetcherFactory()

Source:
  • media/fetcher/ContentFetcherFactory.js, line 15
Example

Usage example:

// Create a content fetcher for the Peacock API
const peacockFetcher = ContentFetcherFactory.createContentFetcher('Peacock', 'https://api.peacocktv.com', {
  Authorization: 'Bearer your_access_token'
});
